파이썬의 웹 프레임 워크

(Micro Framework)
(마이크로 프레임워크) -> 경량화, 소형화..
파이썬: Flask(WSGI 구현체인 Werkzeuge와 템플릿 Jinja2), Bottle
루비 : 시나트라(Sinatra) - 마이크로 프레임워크 원조

마이크로 프레임워크: 웹 프로그래밍에 있어서 가장 핵심적인 요소만을 포함하고 있는 프레임워크

핵심적 기능 이외 개방형 인증기술, 데이터베이스 통합, 유효성 검사를 직접 개발해서 사용할 수 있음

(Full Stack Framework)
풀스택 프레임워크
파이썬 : Django. Web2py, Turbogear
자바 : Spring
루비 : Rails 
풀스택 프레임워크 : 웹프로그래밍을 할 때 필요로 하는 모든 것들을 종합적으로 갖추고 있는 프레임워크

(장점: 핵심적 요소 가지고 있음.. (주어진 환경에 맞춰)새롭게 개발..) / 사용상 편리하지만 자유도가 떨어짐

인증과 권한, ORM,템플릿 라이브러리, 국제화와 지역화, 관리자, 보안 등의 여러요소를 갖춤
